# Конвертер карт глубины в 3D модели

Программа для преобразования карт глубины в трёхмерные модели с визуализацией и экспортом в форматы VRML/WRL.

## Содержимое репозитория

- `depth_converter.py` — основной исходный код на Python
- `DepthMap_Test.dat` — тестовые данные (фрагмент поверхности сферы)
- `DepthMap_12.dat` — входные данные для варианта 12
- `output12.vrml` — результат экспорта в формате VRML
- `output12.wrl` — результат экспорта в формате WRL (альтернативное расширение)

## Описание

Программа выполняет:
1. Чтение бинарных файлов карт глубины (формат: double)
2. Преобразование в трёхмерную треугольную сетку
3. Визуализацию с использованием OpenGL
4. Экспорт в форматы VRML/WRL

## Зависимости

```bash
pip install numpy PyOpenGL PyOpenGL_accelerate
```
## Управление в OpenGL

Стрелки — вращение камеры  
W/S — приближение/отдаление  
R — сброс вида к начальному положению  
Q/Esc — выход из программы 

## Формат входных данных

Карта глубины представляет собой бинарный файл со структурой:  
height (double) — высота матрицы  
width (double) — ширина матрицы  
data (double[width*height]) — массив значений глубины

## Форматы вывода
VRML (.vrml) — основной формат экспорта (если не получается открыть файл с этим форматом в Blender, то можно попробовать на этой веб-странице: https://imagetostl.com/ru/convert/file/vrml/to/blend#convert)  
WRL (.wrl) — альтернативное расширение (совместимо с Blender)
